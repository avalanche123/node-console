# node.js console
node.js application that lets you embed a terminal in the browser.

## dependencies
* node.js
* npm
* modern browser

## installation
* git clone the repository
* cd into the project root
* npm install
* node server.js

## usage
open up localhost:8080/index.html
use the terminal
